## Game Overview
**Dog of Duty** is a tower defense game where the player must prevent mail carriers from delivering their mail by strategically placing dogs along their route to scare them away.

## Project Context
This game was developed during the **Valleyfield Intercollegiate Game Jam 2022**, where it won **2nd place**.

## Tools
This game was developed using **Construct 3**.
