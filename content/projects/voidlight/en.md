## Game Overview  
**Voidlight** is a survival game where the player must mine resources while avoiding monsters lurking in the dark.

## Project Context  
This game was developed during **Wonderjam Winter 2025**, where it won **1st place**.

## Tools  
This game was developed using **Godot 4**.

## My Role
I worked on the resource management and item crafting systems.
