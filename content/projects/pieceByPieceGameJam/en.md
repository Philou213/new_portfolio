## Game Overview  
**Piece by Piece** is a platformer-puzzle game where each level is made up of puzzle pieces that the player must assemble to create a path to the goal.  
The game won Wonderjam Fall 2024 before continuing development as a commercial project.

## Project Context  
This game was developed during the Wonderjam Game Jam at UQAC in Fall 2024. It won **1st place**.

## Tools  
This game was developed using **Godot 4**.
