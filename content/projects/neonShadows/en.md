## Game Overview
**Neon Shadows** is a stealth game where the player must navigate through levels without being detected. However, a sinister clone follows closely behind, mimicking every movement.

## Project Context
This game was developed during **Mini Jam 86: Sports**.

## Tools
This game was developed using **Construct 3**.
