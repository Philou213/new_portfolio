## Game Overview
**Double Time** is a first-person shooter (FPS) where the player must complete levels as quickly as possible. To succeed, the player needs to eliminate enemies efficiently while mastering movement speed and parkour mechanics.

## Project Context
This game was developed for the **Video Games II** course in the **Computer Science Technology** program at **Cégep de Sainte-Foy**.

## Tools
This game was developed using **Unity**.
