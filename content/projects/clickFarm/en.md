## Game Overview
**Click Farm** is an automation-based game where the player must generate as many clicks as possible. To achieve this, they can unlock and use increasingly advanced machines to optimize production.

## Project Context
This game was developed during the **Code Game Jam 2026**. It was created in under 30 hours by a team of three. The game won **2st place**.

## Tools
This game was developed using **Godot 4**.

## My Contributions
I worked on the user interface, menus, and implemented the purchasing and in-game economy systems.
