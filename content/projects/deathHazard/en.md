## Game Overview
**Death Hazard** is a top-down shooter where the player must survive as long as possible against endless waves of zombies.

## Project Context
This game was developed as the final project for the **PEI** program in Grade 11 (Secondary 5).

## Tools
This game was developed using **Godot 3**.
