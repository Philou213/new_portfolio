## Game Overview
**Blind Bond** is an online cooperative game where two players must work together to progress through the levels. The twist: each player can only see the other player's character, never their own.

## Project Context
This game was developed during **Mini Jam 77: Courage**.

## Tools
This game was developed using **Construct**.
