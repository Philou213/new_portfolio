## Game Overview
**Loaded Dice** is a roguelike game where the player must progress as far as possible through their adventure. Throughout the run, they can obtain dice that enhance their attributes. However, both the dice values and the upgrades are entirely random, making luck a key factor.

## Project Context
This game was developed during the **GMTK Game Jam 2022**.

## Tools
This game was developed using **Construct 3**.
