## Game Overview  
**You are a Triangle** is an arcade game where the player must survive as long as possible against waves of squares. Each new wave introduces additional challenges to increase difficulty.

## Project Context  
This game was developed during the **GMTK Game Jam 2020**.

## Tools  
This game was developed using **Construct 3**.

## My Role
I worked on the sounds, music, and the design of various Power-ups.
