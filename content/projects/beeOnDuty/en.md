## Game Overview
**Bee on Duty** is a tower defense game where the player must collect pollen scattered across the map and use it as currency to purchase flowers. These flowers help defend the hive against hostile wasps attempting to attack it.

## Project Context
This game was developed during the **Valleyfield Intercollegiate Game Jam 2023**, where it won **1st place**.

## Tools
This game was developed using **Construct**.
