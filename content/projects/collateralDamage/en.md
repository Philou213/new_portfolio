## Game Overview
**Collateral Damage** is a local multiplayer platform fighter where players face off in a series of intense matches. With a wide variety of weapons and abilities, each player aims to knock opponents out of the arena. Because the environment is destructible, matches grow increasingly chaotic as they progress.

## Project Context
This game was developed as the capstone project for the **Computer Science Technology** program at **Cégep de Sainte-Foy**.

## Tools
This game was developed using **Unity**.
