## Game Overview
**Greening** is an action game where the player confronts an evil flame that must be extinguished by watering it. However, the player must also protect the remaining vegetation—otherwise, everything they are trying to save will be lost.

## Project Context
This game was developed for the **MothoGlacia Construct Reddit Game Jam**, where it won **1st place**.

## Tools
This game was developed using **Construct 3**.
